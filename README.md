# Trailers (aka Filmstrip) App

Movies and TV Series recommendations.

### Build Requirements

- Dart 3.11.x
- Flutter 3.41.x

## Environment variables

| Name                  | Type   | Default                             |
| --------------------- | ------ | ----------------------------------- |
| AD_SIZE               | String | 300x250                             |
| AD_URL                | String | https://ads.mango3.app/300x250.html |
| IDENTITY_API_URL      | String | https://api.id.mango3.app/          |
| IDENTITY_CLIENT_ID    | String |                                     |
| IDENTITY_REDIRECT_URL | String | app.mango3.trailers://oauth         |
| IDENTITY_URL          | String | https://id.mango3.app/              |
| SEARCH_DEBOUNCE       | Number | 500                                 |
| TRAILERS_API_URL      | String | https://api.trailers.mango3.app/    |
| TRAILERS_URL          | String | https://trailers.mango3.app/        |
