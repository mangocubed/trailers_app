# Mango³ Trailers Application

Movies and TV Series recommendations.

### Build Requirements

- Dart 3.11.x
- Flutter 3.41.x

| IDENTITY_API_URL      | String | https://api.id.mango3.app/          |
| IDENTITY_CLIENT_ID    | String |                                     |
| IDENTITY_REDIRECT_URL | String | app.mango3.trailers://oauth         |
| IDENTITY_URL          | String | https://id.mango3.app/              |
| TRAILERS_API_URL      | String | http://api.trailers.mango3.app/     |
| TRAILERS_URL          | String | https://trailers.mango3.app/        |
